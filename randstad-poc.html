Randstad web store

Randstad ecommerce web-store

About This Project

Randstad wanted an internal webstore for its employees so that they can purchase Nike's sports related goodies. In order to provide rewards to its employees users can use the web store to purchase fitness related items. I have currently use the the price (euro) which could be later switched to rewards/points.

Project Details

 http://on.be.net/1z57bbo 

Process

As it was a POC, we wanted to quickly create a POC so that we can quickly visualize the main user flow (product checkout). We didn't have any extended user research as we wanted to create a quick mocks and then proceed further.

As per the project I initially evaluated the existing shop as per the usability guidelines and based upon them I proposed additional functionality to improve the UX of the web shop.

Before proceeding to visual design, I created paper mocks and tested them with the team. Also ensured that I correct the major usability flows in the existing design. As a POC we decided to have a more detailed user research when we implement the design.
